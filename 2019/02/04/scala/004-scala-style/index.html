换行

换行一般是80个字符。

每个连续的行应该从第一行缩进两个空格。 还要记住，Scala要求每个“换行”要么具有未闭合的括号，要么以中缀方法结束，其中没有给出正确的参数：

如果没有这种尾随方式，Scala会在一行的末尾推断出一个分号，它有时会包装，有时甚至不会发出警告而抛弃编译。
